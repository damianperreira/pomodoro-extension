Privacy Policy

Last updated: April 9, 2026

Pomodoro Timer with Music & Tasks is a browser extension that helps you manage focused work sessions. This policy explains what data the extension accesses and how it is handled.

Data stored locally

The extension saves the following data using Chrome's built-in storage (chrome.storage.sync):

  • Task names and completion status
  • Active task selection
  • Daily pomodoro count and focus time

This data syncs across your signed-in Chrome browsers through your Google account. It is never sent to any external server controlled by this extension.

No personal data collection

The extension does not collect, transmit, or store any personal information, browsing history, cookies, or analytics. There is no backend server and no user accounts.

Third-party services

Radio Browser API — When you browse radio stations in the Radio tab, the extension sends a genre search query (e.g. "ambient") to the Radio Browser open-source API. No personal data is included in these requests.

Audio streams — When you play a sound or radio station, your browser connects directly to that station's public stream URL (e.g. SomaFM, MyNoise). The extension does not proxy, log, or monitor these connections.

Notifications

The extension uses Chrome's notification API to alert you when a focus or break session ends. These are local desktop notifications only — no push notification service is used.

Changes to this policy

If this policy is updated, the revision date at the top will be changed. Continued use of the extension after changes constitutes acceptance.
